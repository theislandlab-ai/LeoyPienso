# Cómo contribuir a Leo y Pienso

Gracias por querer aportar. Este proyecto vive porque personas como vos donan su tiempo y talento.

## Tipos de contribución

### 📝 Contenido (textos)
Si sos docente, bibliotecario o simplemente escribís bien:
- Leé la [Guía de contenido](docs/guia-contenido.md)
- Usá el formato JSON documentado en `/schemas/contenido.schema.json`
- Agregá tu texto en `/content/es/`
- Enviá un PR

### 💻 Código
- Buscá issues con la etiqueta `good-first-issue`
- Comentá en el issue si querés tomarlo
- Si es un cambio grande, abrí un issue primero para discutirlo

### 🌍 Traducciones
- Copiá `/content/es/` a `/content/pt/`, `/content/en/`, etc.
- Traducí los textos (respetando el formato JSON)
- Traducí las strings de la UI en `/src/i18n/`

### 🐛 Reportar bugs
- Abrí un issue describiendo: qué pasó, qué esperabas, cómo reproducirlo
- Capturas de pantalla ayudan mucho

## Estándares

- **Código:** seguí el estilo existente (Prettier, ESLint)
- **Commits:** mensajes claros en español o inglés
- **PRs:** describí qué cambia y por qué

## Código de conducta

Sé amable. Este proyecto busca ayudar a niños a leer mejor. Todo aporte desde ese lugar es bienvenido.
